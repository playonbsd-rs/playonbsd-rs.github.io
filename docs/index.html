    playonbsd-rs 

    This a toy project I started to learn a bit of Rust. It aims at providing some libraries or tools to interact with the PlayOnBSD database.

    For now, the project provides:

    • pobsd-parser: a library to easely parse the PlayOnBSD database.

    • pobsd-db: a library to easely query the PlayOnBSD database. Data can be loaded using the pobsd-parser.

    • pobsd-utils: a binary to easely check or export the PlayOnBSD database.

    • pobsd: a TUI to browse the PlayOnBSD database.

    • igdb: a fork of an existing but deprecated library for the API of the IGDB database.
